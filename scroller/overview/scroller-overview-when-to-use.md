Use **Scroller** when a container has overflow content that the user must be able to navigate.

- Inside any vertically or horizontally scrollable container where native browser scrollbar styling is inconsistent or visually misaligned with the UI
- In sidebars, panels, modals, sheets, and drawers with constrained heights and overflow content
- In data-dense surfaces such as tables, lists, or feed views with variable content length
- When a scrollable region needs to visually communicate scroll position and proportion at a glance
